# Repository guidance

## Distributable skills are source artifacts here

- Do not invoke or apply any skill defined under `skills/`, or an installed copy originating from this repository, while editing, testing, reviewing, packaging, or publishing this repository.
- Treat these skills as product source under development, not as repository workflow instructions. Reading them to edit or evaluate their contents does not activate them.
- This exclusion includes repository setup, backwards-compatibility, implementation, commit, Roborev, TDD, structural-change, finishing, and design skills from this collection.
- Apply a repository skill only when the user explicitly asks to exercise that skill as a bounded test. Do not let the tested skill govern unrelated work on its own source repository.
- System-provided skill-authoring guidance may still be used when creating or updating these source artifacts.
